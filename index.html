功能特点

Surfacia 提供了一套完整的工具，用于自动化分析表面原子化学相互作用。

📊

3D 结构生成

将 SMILES 字符串转换为 3D 分子结构（XYZ 格式）。

🔬

计算化学软件集成

支持 Gaussian、XTB 和 Multiwfn 等量子化学计算软件。

⚛️

原子重排

重新排列 XYZ 文件中的原子，以便进行特定的分析。

🧠

机器学习

内置脚本，用于特征提取和使用 XGBoost 和逐步回归进行回归分析。

📈

SHAP 分析

提供 SHAP（SHapley Additive exPlanations）图，用于解释机器学习模型。

🔗

化学家友好

注重生成化学家可解释的结论，便于理解结构-活性关系。

工作流程概述

1

SMILES 到 3D 结构转换

将分子 SMILES 字符串转换为 3D 分子结构（XYZ 格式）。

python scripts/surfacia_main.py smi2xyz --smiles_csv <path_to_smiles_csv>
2

XTB 优化

使用 XTB 对分子结构进行优化。

python scripts/surfacia_main.py xtbopt --input_folder <xyz_directory> --output_folder <output_directory> [--param_file <optional_param_file>]
3

原子重排

重新排列 XYZ 文件中的原子，以便进行特定的分析。

python scripts/surfacia_main.py reorder --element <element_symbol> --input_dir <input_directory> --output_dir <output_directory>
4

提取子结构

从分子中提取特定的子结构。

python scripts/surfacia_main.py extract_substructure --substructure_file <path_to_substructure_file> --input_dir <input_directory> --output_dir <output_directory> --threshold <matching_threshold>
5

XYZ 转换为 Gaussian 输入

将 XYZ 文件转换为 Gaussian 输入文件。

python scripts/surfacia_main.py xyz2gaussian --xyz_folder <xyz_directory> --template_file <path_to_template_file> --output_dir <output_directory>
6

运行 Gaussian 计算

执行 Gaussian 量子化学计算。

python scripts/surfacia_main.py run_gaussian --com_dir <com_file_directory>
7

处理 Multiwfn 输出

处理 Multiwfn 的输出文件，提取分子描述符。

python scripts/surfacia_main.py readmultiwfn --input_dir <input_directory> --output_dir <output_directory> --smiles_target_csv <path_to_smiles_target_csv> --first_matches_csv <path_to_first_matches_csv> --descriptor_option <option_number>
8

机器学习分析

利用提取的特征进行机器学习分析。

python scripts/surfacia_main.py machinelearning --full_csv <path_to_full_csv> --output_dir <ml_output_directory> --nan_handling <nan_handling_option> --epoch <number_of_epochs> --core_num <cpu_cores> --train_test_split_ratio <split_ratio> --step_feat_num <number_of_features>

使用指南

前提条件

  • 安装 Anaconda 或 Miniconda
  • 克隆或下载 Surfacia 到本地
  • 安装 requirements.txt 中列出的依赖项
  • 激活您的 Python 环境（例如：conda activate sympy37）

常规用法

python scripts/surfacia_main.py <task> [options]

可用任务：smi2xyz, xtbopt, reorder, extract_substructure, xyz2gaussian, run_gaussian, readmultiwfn, machinelearning, fchk2matches

附加选项

  • 使用 --config <path_to_config_file> 指定自定义配置文件
  • 对于每个任务的详细帮助，使用：python scripts/surfacia_main.py <task> --help

分子描述符列表

Surfacia 根据选择的 descriptor_option 提取分子描述符：

1. 仅分子属性 (option=1)

提取分子级描述符

基本分子特性

  • 样品名称
  • 原子数量
  • 分子量 (Da)
  • 占据轨道数
  • 等值面积 (Å²)
  • 球形度
  • 体积 (Å³)
  • 密度 (g/cm³)
  • 表面暴露原子数

电子特性

  • HOMO: 最高占据分子轨道能量
  • LUMO: 最低未占据分子轨道能量
  • HOMO-LUMO 能隙
  • 偶极矩
  • 四极矩
  • 八极矩
  • 轨道离域指数 (ODI)

分子大小与形状

  • 最远距离: 两个最远原子之间的距离
  • 分子半径
  • 分子大小 (短轴, 中轴, 长轴)
  • 长轴与总轴之比
  • 长度/直径比
  • 分子平面参数 (MPP)
  • 平面偏差跨度 (SDP)

局部电子特性

  • 局部电子附着能 (LEAE)
  • 静电势 (ESP)
  • 电荷平衡 (ν)
  • 内部电荷分离 (Π)
  • 分子极性指数 (MPI)
  • 极性表面积
  • 平均局部电离能 (ALIE)

2. 分子+特定原子属性 (option=2)

包括 Option 1 的所有描述符，外加特定原子的描述符

特定原子特性

  • Atom1_LEAE_最小值/平均值
  • Atom1_ESP_面积/最小值/最大值/平均值
  • Atom1_ESP_Pi (内部电荷分离)
  • Atom1_ALIE_最小值/最大值

3. 分子+原子+片段属性 (option=3)

包括 Option 2 的所有描述符，外加基于片段索引的片段级描述符

片段级特性

  • Frag_LEAE_最小值/最大值/平均值
  • Frag_ESP_最小值/最大值/表面积/平均值/方差
  • Frag_ESP_Pi (内部电荷分离)
  • Frag_ALIE_最小值/最大值/平均值/方差
